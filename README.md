# RunnerJumper

##Для запуска:

`gulp` в корневой папке<br/>
запустите index.html<br/>

##Как играть:

###Меню

Щелкните по экрану либо нажмите Spacebar чтобы перейти в игру
Клваиша ESC поможет вернуться в меню

###Игра
Стрелка ВВЕРХ - прыжок<br/>
Стрелка ВНИЗ - ускоренное падение<br/>
R - начать сначала<br/>
Счет демонстрирует то, как долго вам удалось прожить<br/>
Наивысший счет демонстрирует самую большую длину жизни<br/>
